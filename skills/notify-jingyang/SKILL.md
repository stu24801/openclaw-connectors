# 通知景揚 Skill

## 使用時機

每當你完成景揚交辦的任務時，**必須**透過此 skill 通知景揚，包括：
- 程式碼修改與部署完成
- 服務重啟完成
- 任何調查、診斷或修復完成
- 長時間背景任務結束

## 通知方式：WhatsApp

直接呼叫內建的 `message` tool，發送訊息給景揚（請至 `USER.md` 查詢景揚的手機號碼作為 target）。

```json
{
  "action": "send",
  "target": "<JINGYANG_PHONE_NUMBER>",
  "message": "✅ [任務摘要，50字以內]\n\n[選填：1-2行重點說明]"
}
```

## 訊息格式範例

```
✅ 文章頁面三欄式 UI 完成

左側章節導覽器 + 右側蝦蝦對話視窗已上線，服務已重啟。
```

```
✅ WRITER_SESSION_KEY 已修正

文章回覆現在會正確送到蝦蝦（main agent），服務已重啟。
```

```
⚠️ 任務遇到問題，需要你決定

[簡述問題 + 你卡住的地方]
```

## 注意事項

- 訊息要**簡短**，景揚在手機上看，50字以內最好
- 只說**做了什麼、結果如何**，不要貼程式碼
- 如果任務失敗或需要景揚做決定，用 ⚠️ 開頭，清楚說明卡在哪裡
- 不需要每個小步驟都通知，**整個任務完成後通知一次**即可
